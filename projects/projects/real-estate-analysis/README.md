# Анализ рынка жилой недвижимости Санкт-Петербурга и Ленинградской области

## Описание проекта

В рамках проекта проведён анализ объявлений о продаже жилой недвижимости в Санкт-Петербурге и городах Ленинградской области.

Цель анализа - определить наиболее перспективные сегменты рынка недвижимости и выявить сезонные особенности размещения объявлений. Результаты исследования могут быть использованы агентством недвижимости для планирования стратегии выхода на региональный рынок.

## Цель исследования

На основе данных объявлений о продаже квартир определить:

- какие типы недвижимости продаются быстрее;
- какие характеристики объектов влияют на скорость продажи;
- существуют ли сезонные колебания активности на рынке недвижимости.

## Основные задачи

В ходе анализа были решены следующие задачи:

- исследовано время активности объявлений о продаже недвижимости;
- определены сегменты объектов, которые продаются быстрее всего;
- проанализированы характеристики квартир (площадь, количество комнат, балконов, стоимость квадратного метра);
- проведено сравнение объектов недвижимости Санкт-Петербурга и городов Ленинградской области;
- изучена сезонность публикации и снятия объявлений;
- проанализировано влияние сезонных факторов на стоимость и характеристики недвижимости.

## Описание данных

Для анализа использованы архивные данные объявлений о продаже квартир за несколько лет.

Данные хранятся в схеме `real_estate` и представлены несколькими взаимосвязанными таблицами.

### Таблица `advertisement`

Содержит информацию об объявлениях о продаже недвижимости.

Основные поля:

- `id` — идентификатор объявления  
- `first_day_exposition` — дата публикации объявления  
- `days_exposition` — длительность размещения объявления на сайте  
- `last_price` — стоимость квартиры  

### Таблица `flats`

Содержит характеристики объектов недвижимости.

Основные параметры:

- `total_area` — общая площадь квартиры  
- `living_area` — жилая площадь  
- `kitchen_area` — площадь кухни  
- `rooms` — количество комнат  
- `floor` — этаж квартиры  
- `floors_total` — этажность дома  
- `ceiling_height` — высота потолков  
- `balcony` — количество балконов  
- `airports_nearest` — расстояние до ближайшего аэропорта  
- `parks_around3000` — количество парков в радиусе 3 км  
- `ponds_around3000` — количество водоёмов в радиусе 3 км  

### Таблица `city`

Содержит справочную информацию о населённых пунктах.

- `city_id` — идентификатор населённого пункта  
- `city` — название города  

### Таблица `type`

Содержит информацию о типе населённого пункта.

- `type_id` — идентификатор типа  
- `type` — тип населённого пункта  

## Используемые инструменты

- SQL  
- DBeaver  
- Yandex DataLens  

## Этапы анализа

1. Первичное исследование структуры базы данных.
2. Анализ длительности активности объявлений.
3. Сегментация объявлений по времени размещения.
4. Сравнение характеристик объектов недвижимости.
5. Анализ сезонности публикации и снятия объявлений.
6. Визуализация результатов в дашборде.

## Результаты

По результатам анализа:

- выявлены сегменты недвижимости, объявления о которых закрываются быстрее;
- определены характеристики квартир, влияющие на скорость продажи;
- обнаружены различия между рынком недвижимости Санкт-Петербурга и Ленинградской области;
- выявлены сезонные колебания активности продавцов и покупателей;
- определены периоды с наибольшим количеством публикаций и закрытий объявлений.
