# Анализ игровой экономики и поведения игроков «Секреты Темнолесья»

## Описание проекта

В рамках проекта проведён анализ игровой экономики для «Секреты Темнолесья». 
Исследование направлено на изучение монетизации игры, поведения игроков при совершении внутриигровых покупок и влияния характеристик персонажей на игровую активность.
Результаты анализа могут быть использованы маркетинговой и продуктовой командами для повышения эффективности монетизации, балансировки игрового процесса и улучшения пользовательского опыта.

## Цель исследования

На основе данных об игроках и внутриигровых покупках определить:
* долю платящих игроков и факторы, влияющие на совершение платежей;
* особенности покупательского поведения пользователей;
* наиболее популярные игровые предметы;
* наличие аномалий в транзакциях;
* влияние расы персонажа на активность игроков и потребность во внутриигровых покупках.

## Основные задачи

В ходе анализа были решены следующие задачи:
* рассчитана доля платящих игроков среди всех зарегистрированных пользователей;
* исследована зависимость вероятности совершения платежа от выбранной расы персонажа;
* рассчитаны основные статистические показатели стоимости внутриигровых покупок;
* выявлены и проанализированы аномальные покупки с нулевой стоимостью;
* проведён анализ популярности эпических предметов;
* выполнена сегментация игроков по игровым расам;
* исследована активность игроков при покупке эпических предметов;
* проверена гипотеза о различиях сложности прохождения игры для разных рас персонажей.

## Описание данных

Для анализа использованы данные игровой базы «Секреты Темнолесья».
База содержит информацию о:
* зарегистрированных игроках;
* характеристиках игровых персонажей;
* внутриигровых покупках эпических предметов;
* операциях покупки игровой валюты за реальные деньги.

Основные анализируемые показатели:
* идентификатор игрока;
* раса персонажа;
* факт совершения платежа;
* стоимость внутриигровой покупки;
* наименование эпического предмета;
* количество покупок игрока.

## Используемые инструменты
* SQL
* DBeaver

## Этапы анализа
1. Исследование структуры базы данных.
2. Анализ платящих игроков и показателей монетизации.
3. Исследование стоимости и структуры внутриигровых покупок.
4. Поиск и анализ аномальных транзакций.
5. Анализ популярности игровых предметов.
6. Сегментация игроков по расам персонажей.
7. Проверка гипотезы о влиянии расы на игровую активность.
8. Формирование рекомендаций для продуктовой и маркетинговой команд.

## Результаты
По результатам анализа:
* доля платящих игроков составила около 18% от общего числа зарегистрированных пользователей;
* существенной зависимости между расой персонажа и вероятностью совершения платежа не выявлено;
* обнаружено более 1,3 млн внутриигровых покупок с высокой вариативностью стоимости;
* выявлены аномальные транзакции с нулевой стоимостью, требующие дополнительной проверки;
* установлено, что около 98% всех покупок приходится на два наиболее популярных эпических предмета;
* выявлены различия в покупательской активности игроков разных рас;
* получены рекомендации по балансировке игрового процесса для снижения различий в сложности прохождения между персонажами разных рас.
