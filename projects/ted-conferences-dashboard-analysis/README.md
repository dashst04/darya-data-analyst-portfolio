## Описание проекта

В рамках проекта был разработан интерактивный аналитический дашборд для исследования данных конференций TED. Заказчиком выступила компания, получившая лицензию на проведение собственных TED-конференций и заинтересованная в использовании опыта предыдущих мероприятий для организации успешного события.

Дашборд позволяет анализировать популярные темы выступлений, характеристики конференций, активность аудитории и особенности выступающих, а также получать детальную информацию по каждому выступлению.

## Цель исследования

На основе исторических данных TED-конференций определить:

* какие темы вызывают наибольший интерес аудитории;
* какие выступления получают максимальный отклик зрителей;
* каковы типичные характеристики успешных конференций;
* какие категории спикеров наиболее востребованы;
* какие параметры необходимо учитывать при планировании будущих мероприятий.

## Основные задачи

В ходе проекта были решены следующие задачи:

* проведён анализ структуры и полноты данных;
* рассчитаны ключевые показатели TED-конференций;
* исследована длительность выступлений и количество спикеров на конференциях;
* проанализированы просмотры, аплодисменты и реакция аудитории;
* определены наиболее популярные темы и теги выступлений;
* выявлены самые смешные выступления по количеству реакций аудитории;
* проведён анализ профессиональной деятельности наиболее успешных спикеров;
* реализована система фильтрации по странам, конференциям, тегам и датам.

## Описание данных

Для анализа использованы данные TED-конференций, представленные в трёх таблицах.

### Таблица events

Содержит информацию о конференциях.

Основные поля:

* conf_id — идентификатор конференции;
* event_name — название конференции;
* country — страна проведения.

### Таблица speakers

Содержит информацию о выступающих.

Основные поля:

* author_id — идентификатор выступающего;
* speaker_name — имя спикера;
* speaker_occupation — профессия или род деятельности;
* speaker_description — описание спикера.

### Таблица talks

Содержит информацию о выступлениях.

Основные поля:

* talk_id — идентификатор выступления;
* title — название выступления;
* duration — длительность выступления;
* views_count — количество просмотров;
* main_tag — основная тема выступления;
* laughter_count — количество реакций смеха;
* applause_count — количество аплодисментов;
* film_date — дата записи выступления.

## Структура дашборда

Дашборд включает:

### Общие показатели

* количество конференций;
* количество выступлений;
* количество уникальных спикеров;
* количество тематических тегов.

### Анализ конференций

* средняя длительность выступлений;
* максимальное количество просмотров;
* суммарное количество аплодисментов;
* количество спикеров на конференциях.

### Анализ выступлений

* топ-10 самых смешных выступлений;
* топ-20 популярных тематических тегов;
* показатели вовлечённости аудитории.

### Анализ спикеров

* наиболее популярные профессии выступающих;
* категории спикеров с наибольшим количеством аплодисментов.

## Результаты

По итогам проекта был разработан интерактивный дашборд, позволяющий исследовать историю TED-конференций на разных уровнях детализации.

Решение помогает выявлять наиболее востребованные темы, анализировать реакцию аудитории, оценивать успешность выступлений и выбирать наиболее подходящих спикеров для будущих мероприятий.
