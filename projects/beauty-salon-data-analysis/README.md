# Создание аналитического отчета для салона красоты

## Описание проекта
Проект по анализу данных салона для кудрявых волос Waves, выполненный в Google Sheets.
Цель проекта — исследовать структуру доходов и расходов салона, проанализировать эффективность промокампаний, а также определить наиболее популярные товары и услуги для поиска точек роста прибыли бизнеса.

## Цели проекта

В рамках проекта были поставлены следующие задачи:
- провести предобработку данных;
- очистить таблицы от ошибок и дубликатов;
- автоматизировать расчёты с помощью функций Google Sheets;
- проанализировать доходы и расходы салона;
- оценить эффективность маркетинговых кампаний;
- определить наиболее популярные товары и услуги;
- визуализировать ключевые показатели бизнеса.

## Используемые инструменты
Инструмменты Google Sheets
Сводные таблицы
Функции SUM, VLOOKUP, IFNA, COUNTIFS, SUMIFS
Формулы

## Данные проекта

Проект основан на данных салона Waves за первые 7 месяцев работы бизнеса.

### Структура данных
Лист **export**
- доходные и расходные операции;
- категории товаров и услуг;
- данные о промокодах;
- информацию о типах оплаты;
- даты транзакций;
- стоимость услуг и товаров.

Лист **promocodes**
- промокоды;
- маркетинговые кампании;
- размер скидок;
- идентификаторы кампаний.

## Предобработка данных
В рамках подготовки данных были выполнены следующие действия:
- удаление дубликатов;
- очистка пробелов и некорректных значений;
- проверка форматов данных;
- обработка пропущенных значений;
- сортировка данных;
- настройка фильтров;
- создание дополнительных вычисляемых столбцов:
    * Вид транзакции (товар/услуга)
    * Месяц транзакции
    * ID промокампании
    * Название промокампании
    * Процент скидки
    * Сумма скидки
    * Итоговая стоимость транзакции

## Выполненный анализ
1. Анализ доходов и расходов
2. Анализ промокампаний
3. Анализ товаров и услуг

## Визуализации
В проекте были построены:
- Столбчатая диаграмма, отражающая динамику доходов и расходов;
- Визуализации эффективности промокампаний;
- Графики динамики спроса на товары и услуги, а также динамика выручки от них;

## Результат проекта
В результате проекта был создан аналитический отчёт, который помогает:
- оценить текущее финансовое состояние салона;
- выявить неэффективные маркетинговые кампании;
- определить наиболее прибыльные товары и услуги;
- найти потенциальные точки роста бизнеса.
