# Анализ рынка вакансий аналитиков данных и системных аналитиков

## Описание проекта

В рамках проекта был проведён анализ базы вакансий с hh.ru с целью изучения текущих тенденций на рынке труда для аналитиков данных и системных аналитиков.

Исследование позволяет понять:
- уровень заработных плат
- распределение вакансий по регионам и компаниям
- требования к опыту работы
- наиболее востребованные профессиональные и дополнительные навыки

## Цель исследования

Проанализировать базу вакансий hh.ru и определить ключевые тенденции рынка труда для аналитиков данных и системных аналитиков.

## Описание данных

В проекте используется таблица `public.parcing_table`, содержащая информацию о вакансиях для аналитиков данных и системных аналитиков.

Данные были предоставлены в готовом виде. Таблица содержит **1801 запись**, каждая из которых соответствует одной вакансии.

В датасете представлены следующие группы признаков:

**Информация о вакансии**
- `id` — уникальный идентификатор вакансии  
- `name` — название вакансии  
- `published_at` — дата публикации вакансии  

**Информация о работодателе**
- `employer` — компания-работодатель  
- `department` — подразделение компании  
- `area` — регион размещения вакансии  

**Требования к кандидату**
- `experience` — требуемый опыт работы  
- `schedule` — график работы  
- `employment` — тип занятости  

**Информация о заработной плате**
- `salary_from` — нижняя граница зарплаты  
- `salary_to` — верхняя граница зарплаты  
- `salary_bin` — категория зарплатной вилки  

**Навыки**
- `key_skills_1–4` — основные профессиональные навыки  
- `soft_skills_1–4` — дополнительные (soft) навыки

## Используемые инструменты

- SQL  
- DBeaver  

## Этапы анализа

- исследование распределения заработных плат
- анализ вакансий по регионам и компаниям
- анализ типов занятости и графиков работы
- анализ распределения грейдов специалистов
- анализ востребованных профессиональных и дополнительных навыков

## Результаты

В ходе анализа были получены следующие выводы:

- Определён диапазон заработных плат, включая средние значения, минимальные и максимальные уровни зарплат.
- Выявлены регионы и компании, в которых сосредоточено наибольшее количество вакансий.
- Проанализированы наиболее распространённые типы занятости и графики работы.
- Изучено распределение грейдов специалистов (Junior, Middle, Senior).
- Определены основные работодатели и предлагаемые условия труда.
- Выявлены наиболее востребованные профессиональные и дополнительные навыки для различных позиций аналитиков.
