# Портфолио проектов по анализу данных

Привет! Меня зовут Дарья.

В этом репозитории собраны мои проекты по анализу данных, выполненные в процессе обучения и практики.

Моя цель — развивать навыки анализа данных, продуктовой аналитики и принятия решений на основе данных.

Инструменты и технологии:
- Python (pandas, numpy, seaborn, matplotlib)
- SQL
- Excel / Google Sheets
- Jupyter Notebook
- DBeaver
- Yandex DataLens

Проекты включают:
- разведочный анализ данных (EDA)
- визуализацию данных
- работу с реальными датасетами
- получение бизнес-инсайтов на основе данных

В репозиторий будут добавляться новые проекты.
