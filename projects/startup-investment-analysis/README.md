# Анализ финансирования стартапов и инвестиционных стратегий

## Описание проекта

В проекте проведён анализ исторических данных о финансировании стартапов. Цель исследования - выявить закономерности привлечения инвестиций, изучить структуру различных типов финансирования и оценить их эффективность с точки зрения возврата инвестиций.

Результаты анализа могут быть использованы инвестиционными компаниями для оценки перспектив различных типов вложений и формирования инвестиционных стратегий.

## Цель исследования

Исследовать структуру и динамику финансирования стартапов, а также определить наиболее эффективные типы инвестиций с точки зрения объёмов вложений и возврата средств.

## Основные задачи

В рамках проекта были выполнены следующие этапы анализа:

- проведена предобработка данных (обработка пропусков, проверка корректности значений, удаление дубликатов);
- проанализирована структура и объёмы финансирования стартапов;
- выделены группы компаний по срокам привлечения инвестиций;
- исследованы различные типы финансирования и их популярность;
- проведён анализ распределения объёмов инвестиций и выявлены аномальные значения;
- изучена динамика инвестиционных раундов и объёмов финансирования по годам;
- рассчитана доля возврата инвестиций для различных типов финансирования;
- сформулированы выводы о наиболее перспективных направлениях инвестирования.

## Описание данных

Для анализа использованы два датасета, содержащие информацию о финансировании стартапов и возврате инвестиций.

### Таблица `cb_investments`

Содержит информацию о компаниях и привлечённых инвестициях.

Основные группы признаков:

**Информация о компании**

- `name` — название компании  
- `homepage_url` — ссылка на сайт  
- `category_list` — категории деятельности компании  
- `market` — основной рынок или отрасль  

**География компании**

- `country_code` — страна  
- `state_code` — регион или штат  
- `region` — регион расположения  
- `city` — город  

**Информация о развитии компании**

- `founded_at` — дата основания  
- `founded_year` — год основания  
- `funding_rounds` — количество раундов финансирования  
- `participants` — количество инвесторов  

**Информация о финансировании**

- `funding_total_usd` — общий объём привлечённых инвестиций  
- `first_funding_at` — дата первого раунда финансирования  
- `last_funding_at` — дата последнего раунда  

**Типы инвестиций**

Данные также содержат распределение инвестиций по различным типам финансирования, включая:

- посевные инвестиции (seed);
- венчурное финансирование (venture);
- инвестиции бизнес-ангелов (angel);
- долговое финансирование (debt financing);
- краудфандинг;
- прямые инвестиции (private equity);
- инвестиции после IPO.

### Таблица `cb_returns`

Содержит информацию о возврате инвестиций по годам и типам финансирования.

Основные поля:

- `year` — год возврата средств  
- показатели возврата инвестиций для различных типов финансирования (seed, venture, angel, debt financing, private equity и другие)

Эти данные позволяют оценить эффективность различных инвестиционных стратегий.

## Этапы анализа

1. Загрузка и первичное исследование данных.
2. Предобработка данных и проверка их корректности.
3. Анализ распределения инвестиций и выявление выбросов.
4. Исследование структуры различных типов финансирования.
5. Анализ динамики инвестиций по годам.
6. Оценка возврата инвестиций для различных типов финансирования.

## Результаты

По итогам исследования:

- выявлены основные тенденции финансирования стартапов;
- определены наиболее распространённые типы инвестиций;
- выявлены различия в объёмах инвестиций между сегментами рынка;
- проведена оценка эффективности различных типов финансирования на основе возврата инвестиций;
- сформулированы рекомендации по возможным инвестиционным стратегиям.
