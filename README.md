# Портфолио проектов по анализу данных

Привет! Меня зовут Дарья.

В этом репозитории собраны мои проекты по анализу данных, выполненные в процессе обучения и практики.

Моя цель - развивать навыки анализа данных, продуктовой аналитики и принятия решений на основе данных.

Инструменты и технологии:
- Python (pandas, numpy, seaborn, matplotlib, scipy)
- SQL
- Excel / Google Sheets
- Jupyter Notebook
- DBeaver
- Yandex DataLens

Проекты включают:
- разведочный анализ данных
- визуализацию данных
- работу с реальными датасетами
- получение бизнес-инсайтов на основе данных

## Проекты

### Анализ данных для салона красоты Waves
Анализ финансовых показателей, промокампаний и спроса на товары и услуги салона красоты в Google Sheets.

### Анализ рынка вакансий аналитиков
Анализ требований работодателей и уровня зарплат.

### Анализ активности доноров DonorSearch
Исследование факторов, влияющих на активность доноров.

### Анализ рынка недвижимости
Исследование сегментов недвижимости Санкт-Петербурга и Ленинградской области.

### Анализ финансирования стартапов
Исследование структуры инвестиций и эффективности различных типов финансирования.

### Анализ игровой экономики MMORPG «Секреты Темнолесья»

Исследование монетизации, внутриигровых покупок и активности игроков с использованием SQL. Анализ доли платящих пользователей, популярности игровых предметов и влияния расы персонажа на игровое поведение.

### Анализ конференций TED и разработка дашборда

Разработка интерактивного дашборда для анализа TED-конференций в SQL и DataLens. Исследование популярных тем, характеристик выступлений, активности аудитории и востребованности спикеров.

В репозиторий также будут добавляться еще и новые проекты.
