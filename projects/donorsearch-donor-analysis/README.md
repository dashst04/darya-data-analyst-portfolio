# Анализ активности доноров проекта DonorSearch

## Описание проекта

В проекте проведён анализ данных платформы DonorSearch — сервиса, который помогает людям становиться донорами крови и делать регулярные донации.

Цель платформы - повышать вовлечённость доноров и стимулировать регулярную сдачу крови. Анализ данных позволяет понять, какие факторы влияют на активность пользователей, а также определить возможные стратегии для увеличения количества донаций.

Проект демонстрирует применение анализа данных в социально значимых инициативах и показывает, как аналитика может помогать развитию благотворительных и медицинских проектов.

## Цель исследования

Выявить факторы, влияющие на активность доноров, и определить возможные направления для повышения вовлечённости пользователей платформы.

## Основные задачи

В рамках проекта были решены следующие аналитические задачи:

- определить регионы с наибольшим количеством зарегистрированных доноров;
- проанализировать динамику количества донаций по месяцам за 2022–2023 годы;
- определить наиболее активных доноров на платформе;
- оценить влияние системы бонусов на активность доноров;
- исследовать привлечение новых доноров через социальные сети;
- сравнить активность однократных доноров и доноров, совершающих повторные донации;
- сопоставить планируемые донации с фактическими для оценки эффективности планирования.

## Описание данных

База данных проекта состоит из нескольких связанных таблиц, содержащих информацию о пользователях платформы, их донациях, бонусах и участии в мероприятиях.

### Таблица `user_anon_data`

Содержит информацию о донорах, зарегистрированных на платформе.

Основные поля:
- `id` — уникальный идентификатор донора  
- `gender` — пол пользователя  
- `birth_date` — дата рождения  
- `region` — регион проживания  
- `blood_type` — группа крови  
- `registration_date` — дата регистрации на сайте  
- `confirmed_donations` — количество подтверждённых донаций  
- `unconfirmed_donations` — неподтверждённые донации  
- `donations_before_registration` — донации до регистрации  
- `last_activity` — последняя активность пользователя на сайте  

Также таблица содержит информацию:
- о готовности донора сдавать различные компоненты крови,
- о количестве сдач различных компонентов,
- о привязке социальных сетей и аккаунтов,
- об использовании бонусов.

### Таблица `donation_anon`

Содержит информацию о совершённых донациях.

Основные поля:
- `user_id` — идентификатор донора  
- `blood_class` — тип донации (кровь или её компоненты)  
- `donation_date` — дата донации  
- `donation_type` — платная или безвозмездная донация  
- `city`, `region`, `country` — место проведения донации  
- `donation_place` — медицинское учреждение  
- `confirmation` — подтверждение справкой  

### Таблица `donation_plan`

Содержит информацию о запланированных донациях.

Основные поля:
- `user_id` — идентификатор донора  
- `blood_class` — тип планируемой донации  
- `donation_date` — дата запланированной донации  
- `plan_date` — дата создания записи о планировании  
- `region` — регион планируемой донации  
- `plan_status` — статус планируемой донации  

### Таблица `user_anon_bonus`

Содержит информацию о бонусах, которые доноры получают за активность.

Основные поля:
- `user_id` — идентификатор донора  
- `bonus_name` — название бонуса  
- `partner` — партнёр, предоставивший бонус  
- `date_of_use` — дата использования бонуса  
- `user_bonus_count` — количество использованных бонусов  

### Дополнительные таблицы

В базе данных также используются вспомогательные таблицы:

- `user_donation` — участие пользователей в донорских мероприятиях  
- `events` — информация о мероприятиях для доноров  
- `bs_data` — данные о центрах переливания крови  

## Используемые инструменты

- SQL  
- DBeaver  

## Результаты

В ходе анализа были получены следующие выводы:

- определены регионы с наибольшим количеством зарегистрированных доноров;
- проанализирована динамика донаций за 2022–2023 годы;
- выявлены наиболее активные доноры платформы;
- исследовано влияние бонусной системы на активность пользователей;
- определены основные каналы привлечения новых доноров через социальные сети;
- выявлены различия в активности однократных и регулярных доноров;
- оценена эффективность планирования донаций на платформе.
